LVDB Simulator is an interactive electrical training project that replicates the behavior of a low-voltage distribution board in a realistic fault simulation environment. It is designed to help users understand how a main isolator, ELCBs, and MCBs work together in an electrical distribution system, and how faults are detected, isolated, and restored during normal and abnormal operation. The simulator provides a hands-on learning experience where users can switch circuits on and off, simulate leakage faults, observe breaker trips, and study the response of each section in real time.

The system is organized around a main isolator feeding three ELCB sections, with each section controlling six MCB circuits for a total of eighteen outgoing circuits. Each ELCB has its own rating and sensitivity, while each MCB represents an individual protected load. Users can select a specific faulty circuit, run the board in normal operation mode, or use randomized blind test mode to challenge fault identification skills. When a faulty circuit is energized, the corresponding ELCB trips automatically, reflecting real protection behavior and reinforcing practical troubleshooting concepts.

The simulator includes live status indicators for voltage, current, and leakage values, giving users clear feedback on system behavior as they interact with the board. An event log records switching actions, fault events, trips, and resets so the full sequence of electrical behavior can be reviewed step by step. Audio feedback adds realism by simulating switching sounds, power hum, and trip responses. Together, these features create a more immersive and educational training environment.

LVDB Simulator is intended for students, technicians, trainers, and anyone learning electrical distribution board operation and fault diagnosis. It is especially useful for understanding protective device coordination, section isolation, fault tracing, and reset procedures in a safe simulated environment. Built as a web-based interactive tool, it combines visual clarity, circuit logic, and fault behavior into a practical training platform for electrical education and power utility style workflows.
